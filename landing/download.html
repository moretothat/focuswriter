Getting started

  1. 1
    Open the downloaded file

    Double-click the .dmg file to mount it.

  2. 2
    Drag to Applications

    Drag FocusWriter to your Applications folder.

  3. 3
    Bypass Gatekeeper (one-time)

    FocusWriter isn't notarized with Apple yet, so on first open you'll see a "FocusWriter is damaged" warning. To fix it, open Terminal (Cmd+Space, type "Terminal") and run:

    xattr -cr /Applications/FocusWriter.app

    This is a one-time step. Apple flags any app not signed with a paid developer certificate — the app itself is fine.

  4. 4
    Start writing

    Open FocusWriter, choose a time or word goal, and lock yourself in. Happy writing!
